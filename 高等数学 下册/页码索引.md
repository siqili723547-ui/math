## 页码索引生成方法

### 扫描版PDF目录提取

当PDF为扫描版（无文本层）时，使用以下方法提取目录：

**步骤1：渲染目录页为图片**
```python
import fitz  # pymupdf
doc = fitz.open("电子书/高等数学 下册 第八版.pdf")
for page_num in [1, 2, 3]:  # 目录通常在第2-4页
    page = doc[page_num]
    pix = page.get_pixmap(matrix=fitz.Matrix(2, 2))  # 2x缩放提高识别率
    pix.save(f"toc_page_{page_num+1}.png")
doc.close()
```

**步骤2：视觉模型识别**
用 `analyze_image` 工具识别图片中的目录文字，提取章节名称和页码。

**步骤3：计算页码偏移**
- 找到第一章起始页（书本页码1对应的PDF页码）
- 偏移量 = PDF页码 - 书本页码
- 本PDF偏移量 = 7

**步骤4：生成索引表**
- 每小节起始页 = 上一小节结束页（首尾衔接）
- PDF页码 = 书本页码 + 偏移量
