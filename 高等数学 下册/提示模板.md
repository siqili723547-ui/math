# AI笔记生成提示模板

## 使用说明

使用 AI（如 Claudian）根据以下模板生成学习笔记。

---

## ⚠️ 文件命名规范（重要）

AI 生成笔记时**必须**遵循以下命名规则：

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| 章笔记 | `第X章 章名.md` | `第八章 空间解析几何.md` |
| 节笔记 | `第X章/X.X 节名.md` | `第八章/8.1 向量及其线性运算.md` |
| 小节笔记 | `第X章/X.X.X 小节名.md` | `第八章/8.1.1 向量概念.md` |

---

## 基础提示词

```
请根据《高等数学 下册》（同济 第八版）课本，为 [[当前章节]] 生成学习笔记。

要求：
1. 按照目录结构填写每个小节的内容
2. 包含定义、定理、公式
3. 给出典型例题和解题思路
4. 标注重点和难点
5. 补充学习技巧和记忆方法
6. 文件命名必须用编号：如 "8.1.1 向量概念.md"
```

---

## 📋 直接复制使用的指令

### 生成单个小节笔记
```
请为《高等数学 下册》生成 "8.1.1 向量概念" 的笔记。

要求：
1. 创建文件：高等数学 下册/笔记/第八章/8.1 向量及其线性运算/8.1.1 向量概念.md
2. 内容包括：定义、性质、例题、易错点
3. 使用 callout 格式标注重点
4. 文件名必须包含编号 "8.1.1"
```

### 批量生成章节笔记
```
请为《高等数学 下册》第八章生成完整笔记。

要求：
1. 在 "高等数学 下册/笔记/第八章/" 文件夹下创建
2. 每个小节单独一个文件，命名格式：X.X.X 小节名.md
3. 内容包含定义、定理、例题
```

---

## Callout 格式参考

```markdown
> [!definition] 定义
> 定义内容

> [!theorem] 定理
> 定理内容

> [!tip] 学习技巧
> 技巧内容

> [!warning] 易错点
> 注意事项

> [!example] 例题
> 例题内容
```
