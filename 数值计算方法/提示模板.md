# AI笔记生成提示模板

## 使用说明

使用 AI（如 Claudian）根据以下模板生成学习笔记。

---

## ⚠️ 文件命名规范

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| 章笔记 | `第X章 章名.md` | `第三章 非线性方程求根.md` |
| 节笔记 | `第X章/X.X 节名.md` | `第三章/3.1 二分法.md` |
| 小节笔记 | `第X章/X.X.X 小节名.md` | `第三章/3.1.1 二分法原理.md` |

---

## 基础提示词

```
请根据《数值计算方法（第二版）》课本，为 [[当前章节]] 生成学习笔记。

要求：
1. 重点描述算法思想、步骤、误差分析
2. 包含算法伪代码或流程
3. 给出典型例题和计算过程
4. 分析算法的收敛性和复杂度
5. 文件命名必须用编号
```

---

## 生成算法类笔记
```
请为《数值计算方法》生成 "3.1.1 二分法原理" 的笔记。

要求：
1. 创建文件：数值计算方法/笔记/第三章/3.1 二分法/3.1.1 二分法原理.md
2. 内容包括：算法思想、步骤、收敛性分析、误差估计、例题
3. 使用 callout 格式标注
4. 突出数值计算特点
```
