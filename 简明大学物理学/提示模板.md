# AI笔记生成提示模板

## 使用说明

使用 AI（如 Claudian）根据以下模板生成学习笔记。

---

## ⚠️ 文件命名规范

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| 章笔记 | `第X章 章名.md` | `第一章 质点运动学.md` |
| 节笔记 | `第X章/X.X 节名.md` | `第一章/1.1 参考系和质点.md` |
| 小节笔记 | `第X章/X.X.X 小节名.md` | `第一章/1.1.1 参考系.md` |

---

## 基础提示词

```
请根据《简明大学物理学》课本，为 [[当前章节]] 生成学习笔记。

要求：
1. 按照目录结构填写每个小节的内容
2. 包含物理概念、定律、公式
3. 给出典型例题和解题思路
4. 注重物理意义的解释
5. 文件命名必须用编号
```

---

## 生成单个小节笔记
```
请为《简明大学物理学》生成 "1.1.1 参考系" 的笔记。

要求：
1. 创建文件：简明大学物理学/笔记/第一章/1.1 参考系和质点/1.1.1 参考系.md
2. 内容包括：概念定义、物理意义、例题
3. 使用 callout 格式标注
4. 突出物理图像
```
