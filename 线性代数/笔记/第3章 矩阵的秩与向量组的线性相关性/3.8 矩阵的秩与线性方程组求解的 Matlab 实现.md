## 一、矩阵秩的计算

### 1.1 rank 函数

> [!tip] 基本用法
> ```matlab
> r = rank(A)    % 计算矩阵 A 的秩
> ```

> [!example] 例1：计算秩
> ```matlab
> A = [1 2 3; 4 5 6; 7 8 9];
> r = rank(A)    % → 2
>
> B = [1 0 0; 0 1 0; 0 0 1];
> r = rank(B)    % → 3（满秩）
>
> C = [1 2; 2 4];
> r = rank(C)    % → 1
> ```

### 1.2 判断可逆性

> [!example] 例2：判断矩阵可逆
> ```matlab
> A = [1 2; 3 4];
> n = size(A, 1);
>
> if rank(A) == n
>     disp('矩阵可逆')
>     A_inv = inv(A);
> else
>     disp('矩阵不可逆')
> end
> ```

### 1.3 判断线性相关性

> [!example] 例3：判断向量组线性相关性
> ```matlab
> % 向量组（作为列向量）
> a1 = [1; 2; 3];
> a2 = [2; 4; 6];
> a3 = [1; 1; 1];
>
> A = [a1, a2, a3];
> m = 3;  % 向量个数
>
> if rank(A) < m
>     disp('线性相关')
> else
>     disp('线性无关')
> end
> % 输出：线性相关（因为 a2 = 2*a1）
> ```

## 二、线性方程组的求解

### 2.1 左除运算符 `\`

> [!tip] 基本用法
> ```matlab
> x = A \ b    % 解 Ax = b
> ```
>
> - 若有唯一解：返回唯一解
> - 若无解：返回最小二乘解
> - 若有无穷多解：返回一个解（自由变量设为0）

> [!example] 例4：解方程组
> ```matlab
> A = [2 1 -1; 1 -1 2; 3 2 1];
> b = [3; 1; 8];
>
> x = A \ b
> % → [1; 2; 1]
>
> % 验证
> A * x - b    % → [0; 0; 0]
> ```

### 2.2 rref 函数（行最简形）

> [!tip] 基本用法
> ```matlab
> R = rref(A)           % 行最简形
> [R, jb] = rref(A)     % jb 是主元所在列的索引
> ```

> [!example] 例5：求行最简形和秩
> ```matlab
> A = [1 2 3; 4 5 6; 7 8 9];
> [R, jb] = rref(A);
>
> disp('行最简形:'); disp(R);
> disp('主元列:'); disp(jb);
> disp('秩:'); disp(length(jb));
> ```

### 2.3 判断解的情况

> [!tip] 完整判断函数
> ```matlab
> function solve_linear_system(A, b)
> % 判断并求解线性方程组 Ax = b
>
>     [m, n] = size(A);
>     Ab = [A, b];  % 增广矩阵
>
>     rA = rank(A);
>     rAb = rank(Ab);
>
>     fprintf('r(A) = %d, r(A|b) = %d, n = %d\n', rA, rAb, n);
>
>     if rA < rAb
>         disp('无解');
>     elseif rA == n
>         disp('唯一解:');
>         x = A \ b;
>         disp(x');
>     else
>         fprintf('无穷多解（自由变量 %d 个）\n', n - rA);
>         x = A \ b;
>         disp('一个特解:');
>         disp(x');
>     end
> end
> ```

## 三、求基础解系

### 3.1 使用 null 函数

> [!tip] 基本用法
> ```matlab
> Z = null(A)           % 基础解系（标准正交基）
> Z = null(A, 'r')      % 基础解系（有理形式）
> ```

> [!example] 例6：求基础解系
> ```matlab
> A = [1 2 2 1; 2 1 -2 -2; 1 -1 -4 -3];
>
> % 方法1：标准正交基
> Z1 = null(A)
>
> % 方法2：有理形式（更直观）
> Z2 = null(A, 'r')
>
> % 验证
> A * Z2    % → 零矩阵
> ```

### 3.2 使用 rref 手动求解

> [!tip] 手动求基础解系
> ```matlab
> function Z = null_manual(A)
> % 手动求基础解系
>
>     [m, n] = size(A);
>     [R, jb] = rref(A);
>     r = length(jb);        % 秩
>     free = setdiff(1:n, jb); % 自由变量位置
>
>     Z = zeros(n, n-r);
>     for i = 1:(n-r)
>         Z(free(i), i) = 1;
>         Z(jb, i) = -R(1:r, free(i));
>     end
> end
> ```

> [!example] 例7：完整求解过程
> ```matlab
> A = [1 1 -3 -1; 3 -1 -3 4; 1 5 -9 -8];
>
> % 化为行最简形
> [R, jb] = rref(A);
>
> % 秩和自由变量
> r = length(jb);
> n = size(A, 2);
> free = setdiff(1:n, jb);
>
> fprintf('秩 = %d, 自由变量 = %d 个\n', r, n-r);
> fprintf('主元列: '); disp(jb);
> fprintf('自由变量列: '); disp(free);
>
> % 基础解系
> Z = null(A, 'r');
> disp('基础解系:'); disp(Z);
> ```

## 四、求非齐次方程组的通解

### 4.1 完整求解函数

> [!tip] 非齐次方程组通解
> ```matlab
> function [x0, Z] = solve_nonhomogeneous(A, b)
> % 求非齐次方程组 Ax = b 的通解
> % 返回：x0 - 特解, Z - 基础解系
> % 通解形式：x = x0 + Z*c（c 为任意向量）
>
>     [m, n] = size(A);
>     Ab = [A, b];
>
>     rA = rank(A);
>     rAb = rank(Ab);
>
>     if rA < rAb
>         error('方程组无解');
>     end
>
>     % 特解（自由变量设为0）
>     x0 = zeros(n, 1);
>     x0(1:rA) = A(:, 1:rA) \ b;
>
>     % 基础解系
>     Z = null(A, 'r');
>
>     fprintf('特解: '); disp(x0');
>     fprintf('基础解系:\n'); disp(Z');
>     fprintf('通解: x = x0 + Z*c\n');
> end
> ```

### 4.2 使用示例

> [!example] 例8：求非齐次方程组通解
> ```matlab
> A = [1 -1 -1 1; 1 -1 1 -3; 1 -1 -2 3];
> b = [0; 1; -0.5];
>
> % 方法1：自定义函数
> [x0, Z] = solve_nonhomogeneous(A, b);
>
> % 方法2：使用 rref
> Ab = [A, b];
> R = rref(Ab);
> disp('增广矩阵的行最简形:'); disp(R);
>
> % 从行最简形读出解
> ```

## 五、向量组的极大无关组

### 5.1 使用 rref 求极大无关组

> [!example] 例9：求极大无关组
> ```matlab
> % 向量作为列向量
> a1 = [1; 2; 1];
> a2 = [2; 4; 2];
> a3 = [1; 2; 3];
> a4 = [3; 6; 4];
>
> A = [a1, a2, a3, a4];
>
> % 求主元列
> [R, jb] = rref(A);
>
> fprintf('极大无关组的列索引: '); disp(jb);
> fprintf('极大无关组为: a%d, a%d\n', jb);
>
> % 秩
> fprintf('秩 = %d\n', length(jb));
>
> % 用极大无关组表示其他向量
> disp('行最简形:'); disp(R);
> % 从 R 中读出表示式
> ```

## 六、综合示例

> [!example] 例10：完整分析线性方程组
> ```matlab
> % 定义方程组
> A = [1 1 1 1; 1 2 1 2; 2 3 2 3];
> b = [3; 5; 8];
>
> disp('=== 系数矩阵分析 ===');
> fprintf('秩 r(A) = %d\n', rank(A));
> fprintf('增广矩阵秩 r(A|b) = %d\n', rank([A, b]));
> fprintf('未知数个数 n = %d\n', size(A, 2));
>
> disp(' ');
> disp('=== 解的判定 ===');
> if rank(A) < rank([A, b])
>     disp('无解');
> elseif rank(A) == size(A, 2)
>     disp('唯一解');
>     x = A \ b;
>     disp('解:'); disp(x');
> else
>     fprintf('无穷多解，自由变量 %d 个\n', size(A, 2) - rank(A));
>
>     disp(' ');
>     disp('=== 特解 ===');
>     x0 = A \ b;
>     disp(x0');
>
>     disp(' ');
>     disp('=== 基础解系 ===');
>     Z = null(A, 'r');
>     disp(Z');
>
>     disp(' ');
>     disp('=== 通解 ===');
>     disp('x = x0 + c1*xi1 + c2*xi2 + ...');
> end
> ```

## 八、易错点

> [!danger] 常见错误
>
> 1. **`\` 返回的解**：有无穷多解时只返回一个解
> 2. **null 函数**：默认返回正交基，用 `'r'` 获取有理形式
> 3. **精度问题**：`rank` 使用数值方法，可能有微小误差
> 4. **列向量**：向量组作为矩阵的**列**（不是行）
> 5. **rref 主元列**：`jb` 给出的是原矩阵的列索引
