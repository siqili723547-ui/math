## 一、矩阵的创建

### 1.1 直接输入

> [!tip] 矩阵输入语法
> - 用**方括号** `[]` 定义矩阵
> - **行内元素**用空格或逗号 `,` 分隔
> - **行与行**用分号 `;` 或换行分隔

```matlab
% 直接输入矩阵
A = [1 2 3; 4 5 6; 7 8 9]

% 等价写法
A = [1, 2, 3;
     4, 5, 6;
     7, 8, 9]
```

### 1.2 特殊矩阵

```matlab
% 零矩阵
Z = zeros(3, 4)      % 3×4 零矩阵

% 单位矩阵
E = eye(3)           % 3阶单位矩阵
E = eye(3, 4)        % 3×4，主对角线为1

% 全1矩阵
O = ones(2, 3)       % 2×3 全1矩阵

% 随机矩阵
R = rand(3, 3)       % 3×3 均匀分布随机矩阵（0~1）
R = randn(3, 3)      % 3×3 标准正态分布随机矩阵

% 对角矩阵
D = diag([1, 2, 3])  % diag(1,2,3)

% 从向量提取对角元
v = diag(A)          % 提取 A 的主对角元
```

### 1.3 矩阵的生成

```matlab
% 等差序列
x = 1:5              % [1, 2, 3, 4, 5]
x = 1:2:10           % [1, 3, 5, 7, 9]（步长为2）

% 线性等分
x = linspace(0, 1, 5) % [0, 0.25, 0.5, 0.75, 1]
```

## 二、矩阵的基本运算

### 2.1 加减法与数乘

```matlab
A = [1 2; 3 4];
B = [5 6; 7 8];

% 加减法
C = A + B            % 矩阵加法
C = A - B            % 矩阵减法

% 数乘
C = 2 * A            % 标量乘矩阵
C = A * 3            % 标量乘矩阵
```

### 2.2 矩阵乘法

```matlab
A = [1 2; 3 4];
B = [5 6; 7 8];

% 矩阵乘法
C = A * B            % 矩阵乘积

% 矩阵幂
C = A^2              % A² = A * A
C = A^3              % A³
```

### 2.3 矩阵与数组运算的区别（重点 ★）

> [!warning] 矩阵运算 vs 数组运算
> Matlab 区分**矩阵运算**和**数组运算**（点运算）：
>
> | 运算 | 矩阵运算 | 数组运算（点运算） |
> |:----:|:--------:|:-----------------:|
> | 乘法 | `A * B` | `A .* B`（对应元素相乘） |
> | 除法 | `A / B` | `A ./ B`（对应元素相除） |
> | 幂 | `A ^ 2` | `A .^ 2`（每个元素平方） |

```matlab
A = [1 2; 3 4];
B = [2 1; 1 2];

% 矩阵乘法
C = A * B            % → [4 5; 10 14]

% 数组乘法（对应元素）
C = A .* B           % → [2 2; 3 8]

% 数组幂（每个元素平方）
C = A .^ 2           % → [1 4; 9 16]
```

### 2.4 转置

```matlab
A = [1 2 3; 4 5 6];

% 转置
B = A'               % 共轭转置（实数即普通转置）
B = A.'              % 普通转置（不取共轭）
```

## 三、矩阵的分解与提取

### 3.1 提取元素

```matlab
A = [1 2 3; 4 5 6; 7 8 9];

% 单个元素
a = A(2, 3)          % 第2行第3列元素 → 6

% 提取行
row1 = A(1, :)       % 第1行 → [1, 2, 3]
row2 = A(2, :)       % 第2行

% 提取列
col1 = A(:, 1)       % 第1列 → [1; 4; 7]
col2 = A(:, 2)       % 第2列

% 子矩阵
sub = A(1:2, 2:3)    % 第1-2行，第2-3列 → [2 3; 5 6]
sub = A([1,3], [1,3]) % 第1、3行，第1、3列 → [1 3; 7 9]
```

### 3.2 修改元素

```matlab
A = [1 2 3; 4 5 6; 7 8 9];

% 修改单个元素
A(1, 1) = 0          % 第1行第1列改为0

% 修改整行
A(2, :) = [0 0 0]    % 第2行全改为0

% 修改整列
A(:, 3) = 1          % 第3列全改为1
```

### 3.3 矩阵拼接

```matlab
A = [1 2; 3 4];
B = [5 6; 7 8];

% 水平拼接（列增加）
C = [A, B]           % 或 C = horzcat(A, B)
% → [1 2 5 6; 3 4 7 8]

% 垂直拼接（行增加）
C = [A; B]           % 或 C = vertcat(A, B)
% → [1 2; 3 4; 5 6; 7 8]
```

## 四、逆矩阵与线性方程组

### 4.1 求逆矩阵

```matlab
A = [1 2; 3 4];

% 方法1：inv函数
A_inv = inv(A)       % 逆矩阵

% 方法2：幂运算
A_inv = A^(-1)       % 等价于 inv(A)
```

### 4.2 行列式

```matlab
A = [1 2; 3 4];
d = det(A)           % 行列式 → -2
```

### 4.3 解线性方程组

> [!important] 解 $\boldsymbol{Ax} = \boldsymbol{b}$

```matlab
A = [3 2 -1; 2 -2 4; -1 0.5 -1];
b = [1; -2; 0];

% 方法1：逆矩阵
x = inv(A) * b

% 方法2：左除运算（推荐，更高效）
x = A \ b            % 高斯消元法求解

% 方法3：linsolve
x = linsolve(A, b)
```

> [!tip] 推荐使用 `A \ b`
> - `A \ b` 比 `inv(A) * b` 更**高效**且**数值稳定**
> - Matlab 会自动选择最优算法（LU分解等）

### 4.4 矩阵方程

```matlab
% 解 AX = B
X = A \ B            % X = A^{-1}B

% 解 XA = B
X = B / A            % X = BA^{-1}

% 解 AXB = C
X = A \ C / B        % X = A^{-1}CB^{-1}
```

## 五、初等变换与矩阵分析

### 5.1 初等变换的实现

```matlab
A = [1 2 3; 4 5 6; 7 8 9];

% 交换第1、3行
A([1,3], :) = A([3,1], :)

% 第2行乘以2
A(2, :) = 2 * A(2, :)

% 第3行加上第1行的2倍
A(3, :) = A(3, :) + 2 * A(1, :)
```

### 5.2 行阶梯形

```matlab
A = [1 2 3; 4 5 6; 7 8 9];

% 行阶梯形（rref = Reduced Row Echelon Form）
R = rref(A)          % 行最简形
```

### 5.3 矩阵的秩

```matlab
A = [1 2 3; 4 5 6; 7 8 9];
r = rank(A)          % 秩 → 2
```

## 六、矩阵分解（选学）

### 6.1 LU 分解

```matlab
A = [4 3; 6 3];

% LU分解：A = LU
[L, U] = lu(A)
% L: 下三角矩阵
% U: 上三角矩阵

% 带置换的LU分解：PA = LU
[L, U, P] = lu(A)
% P: 置换矩阵
```

### 6.2 QR 分解

```matlab
A = [1 2; 3 4; 5 6];

% QR分解：A = QR
[Q, R] = qr(A)
% Q: 正交矩阵
% R: 上三角矩阵
```

### 6.3 特征值分解

```matlab
A = [1 2; 3 4];

% 特征值与特征向量
[V, D] = eig(A)
% V: 特征向量组成的矩阵
% D: 特征值组成的对角矩阵

% 只求特征值
e = eig(A)           % 返回特征值向量
```

## 八、典型示例

> [!example] 例1：矩阵运算综合
> ```matlab
> % 定义矩阵
> A = [1 2 3; 4 5 6];
> B = [7 8; 9 10; 11 12];
>
> % 矩阵乘法
> C = A * B           % → [58 64; 139 154]
>
> % 转置
> At = A'             % → [1 4; 2 5; 3 6]
>
> % 行列式（需方阵）
> D = [1 2; 3 4];
> det(D)              % → -2
> ```

> [!example] 例2：解线性方程组
> 解方程组：
> $$\begin{cases} x_1 + 2x_2 + 3x_3 = 1 \\ 2x_1 + 5x_2 + 7x_3 = 2 \\ 3x_1 + 7x_2 + 10x_3 = 4 \end{cases}$$
>
> ```matlab
> A = [1 2 3; 2 5 7; 3 7 10];
> b = [1; 2; 4];
>
> % 求解
> x = A \ b           % → [1; -1; 1]
>
> % 验证
> A * x - b           % → [0; 0; 0]（近似）
> ```

> [!example] 例3：矩阵求逆与验证
> ```matlab
> A = [1 2 3; 0 1 4; 5 6 0];
>
> % 求逆
> A_inv = inv(A)
>
> % 验证
> A * A_inv            % → 单位矩阵
> det(A)               % → 1（可逆）
> ```

## 九、注意事项

> [!warning] 常见问题
>
> 1. **矩阵维度匹配**：乘法要求列数=行数
> 2. **点运算**：`.*`, `./`, `.^` 是逐元素运算，不要漏掉点
> 3. **奇异矩阵**：`det(A) = 0` 时 `inv(A)` 会报错或结果不准
> 4. **数值精度**：浮点运算有误差，`A * inv(A)` 可能不是精确的单位阵
> 5. **左除右除**：`\` 是左除，`/` 是右除，方向不同
