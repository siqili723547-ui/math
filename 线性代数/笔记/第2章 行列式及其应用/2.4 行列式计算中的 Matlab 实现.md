## 一、行列式的计算

### 1.1 基本函数

> [!tip] det 函数
> ```matlab
> d = det(A)    % 计算方阵 A 的行列式
> ```

> [!example] 例1：计算行列式
> ```matlab
> A = [1 2; 3 4];
> d = det(A)    % → -2
>
> B = [1 2 3; 4 5 6; 7 8 10];
> d = det(B)    % → -3
> ```

### 1.2 特殊矩阵的行列式

> [!example] 例2：特殊矩阵
> ```matlab
> % 单位矩阵
> E = eye(4);
> det(E)        % → 1
>
> % 对角矩阵
> D = diag([2, 3, 4]);
> det(D)        % → 24 = 2×3×4
>
> % 三角矩阵
> U = [1 2 3; 0 4 5; 0 0 6];
> det(U)        % → 24 = 1×4×6
> ```

### 1.3 行列式与可逆性

> [!example] 例3：判断矩阵可逆性
> ```matlab
> A = [1 2 3; 4 5 6; 7 8 9];
>
> if det(A) ~= 0
>     disp('矩阵可逆')
>     A_inv = inv(A);
> else
>     disp('矩阵不可逆（奇异矩阵）')
> end
> % 输出：矩阵不可逆（奇异矩阵）
> ```

## 二、克拉默法则的 Matlab 实现

### 2.1 基本实现

> [!example] 例4：克拉默法则解方程组
> 解 $\begin{cases} 2x + y = 5 \\ 3x + 2y = 8 \end{cases}$
>
> ```matlab
> % 系数矩阵
> A = [2 1; 3 2];
>
> % 常数向量
> b = [5; 8];
>
> % 系数行列式
> D = det(A);    % → 1
>
> % 计算 D1, D2
> D1 = det([b, A(:,2)]);    % 第1列换成 b
> D2 = det([A(:,1), b]);    % 第2列换成 b
>
> % 解
> x = D1 / D      % → 2
> y = D2 / D      % → 1
> ```

### 2.2 通用函数

> [!tip] 克拉默法则通用实现
> ```matlab
> function x = cramer(A, b)
> % CRAMER 使用克拉默法则解线性方程组 Ax = b
> % 输入：A - 系数矩阵（n×n）
> %       b - 常数向量（n×1）
> % 输出：x - 解向量
>
>     n = size(A, 1);
>     D = det(A);
>
>     if abs(D) < 1e-10
>         error('系数行列式为零，无法使用克拉默法则');
>     end
>
>     x = zeros(n, 1);
>     for j = 1:n
>         Aj = A;
>         Aj(:, j) = b;    % 第 j 列换成 b
>         x(j) = det(Aj) / D;
>     end
> end
> ```

### 2.3 使用示例

> [!example] 例5：三元方程组
> ```matlab
> A = [2 1 -1; 1 -1 2; 3 2 1];
> b = [3; 1; 8];
>
> x = cramer(A, b)
> % → [1; 2; 1]
>
> % 验证
> A * x - b       % → [0; 0; 0]
> ```

## 三、伴随矩阵

### 3.1 伴随矩阵计算

> [!tip] 伴随矩阵公式
> $\boldsymbol{A}^* = |\boldsymbol{A}| \cdot \boldsymbol{A}^{-1}$（当 $\boldsymbol{A}$ 可逆）

```matlab
function As = adjoint(A)
% ADJOINT 计算伴随矩阵
% 输入：A - n×n 方阵
% 输出：As - 伴随矩阵

    n = size(A, 1);
    d = det(A);

    if abs(d) < 1e-10
        % 行列式为零，逐个计算代数余子式
        As = zeros(n);
        for i = 1:n
            for j = 1:n
                % 余子式
                M = A;
                M(i, :) = [];
                M(:, j) = [];
                % 代数余子式
                As(j, i) = (-1)^(i+j) * det(M);
            end
        end
    else
        % 利用公式 A* = |A| * A^{-1}
        As = d * inv(A);
    end
end
```

### 3.2 使用示例

> [!example] 例6：计算伴随矩阵
> ```matlab
> A = [1 2 3; 0 1 4; 5 6 0];
>
> % 方法1：自定义函数
> As = adjoint(A)
>
> % 方法2：利用公式
> det(A) * inv(A)
>
> % 验证 A * A* = |A| * E
> A * adjoint(A) - det(A) * eye(3)
> % → 0 矩阵
> ```

## 四、行列式性质验证

### 4.1 转置性质

> [!example] 验证 $|\boldsymbol{A}^{\mathrm{T}}| = |\boldsymbol{A}|$
> ```matlab
> A = rand(4, 4);
>
> det(A)          % 原行列式
> det(A')         % 转置后
>
> % 两者相等
> abs(det(A) - det(A')) < 1e-10   % → true
> ```

### 4.2 乘积性质

> [!example] 验证 $|\boldsymbol{AB}| = |\boldsymbol{A}| \cdot |\boldsymbol{B}|$
> ```matlab
> A = rand(3, 3);
> B = rand(3, 3);
>
> left = det(A * B);
> right = det(A) * det(B);
>
> abs(left - right) < 1e-10   % → true
> ```

### 4.3 数乘性质

> [!example] 验证 $|k\boldsymbol{A}| = k^n|\boldsymbol{A}|$
> ```matlab
> A = rand(3, 3);
> k = 2;
> n = 3;
>
> left = det(k * A);
> right = k^n * det(A);
>
> abs(left - right) < 1e-10   % → true
> % 注意：不是 k * det(A)
> ```

### 4.4 伴随矩阵性质

> [!example] 验证 $\boldsymbol{A}\boldsymbol{A}^* = |\boldsymbol{A}|\boldsymbol{E}$
> ```matlab
> A = rand(3, 3);
> As = adjoint(A);
>
> left = A * As;
> right = det(A) * eye(3);
>
> norm(left - right) < 1e-10  % → true
> ```

## 五、行列式应用实例

### 5.1 判断线性相关性

> [!example] 例7：判断向量组是否线性相关
> ```matlab
> % 三个三维向量
> v1 = [1; 2; 3];
> v2 = [2; 4; 6];
> v3 = [1; 1; 1];
>
> % 组成矩阵
> A = [v1, v2, v3];
>
> if abs(det(A)) < 1e-10
>     disp('线性相关')
> else
>     disp('线性无关')
> end
> % 输出：线性相关（v2 = 2*v1）
> ```

### 5.2 求解参数问题

> [!example] 例8：确定参数值
> 问 $\lambda$ 为何值时，方程组有非零解？
>
> ```matlab
> syms lambda
>
> A = [lambda+1, 1, 1;
>      1, lambda+1, 1;
>      1, 1, lambda+1];
>
> D = det(A);
>
> % 求解 D = 0
> solve(D, lambda)
> % → 0, -3
> ```

### 5.3 范德蒙德行列式

> [!example] 例9：计算范德蒙德行列式
> ```matlab
> x = [1, 2, 3, 4];
> n = length(x);
>
> % 构造范德蒙德矩阵
> V = ones(n);
> for i = 1:n
>     for j = 1:n
>         V(i, j) = x(j)^(i-1);
>     end
> end
>
> % 或使用 vander 函数（注意：vander 是反的）
> V = fliplr(vander(x'));
>
> det(V)
> % → 12 = (2-1)(3-1)(4-1)(3-2)(4-2)(4-3)
> ```

## 六、数值精度问题

### 6.1 浮点误差

> [!warning] 数值精度问题
> Matlab 使用浮点数计算，行列式结果可能有微小误差。
>
> ```matlab
> A = [1, 2, 3; 4, 5, 6; 7, 8, 9];
> det(A)      % 理论上是 0
> % → 9.1593e-16（非常接近 0，但不等于 0）
> ```

### 6.2 判断奇异矩阵

> [!tip] 使用容差判断
> ```matlab
> function is_singular = isSingular(A)
> % 判断矩阵是否奇异
>
>     tol = 1e-10;  % 容差
>     is_singular = abs(det(A)) < tol;
> end
> ```

### 6.3 更稳健的方法

> [!tip] 使用 rank 或 cond 判断
> ```matlab
> A = [1 2 3; 4 5 6; 7 8 9];
>
> % 方法1：用秩
> rank(A) < size(A, 1)   % → true（奇异）
>
> % 方法2：用条件数
> cond(A) > 1e10         % → true（接近奇异）
>
> % 方法3：用 rcond（推荐）
> rcond(A) < eps         % → true（奇异）
> ```

## 八、完整示例：克拉默法则

> [!example] 完整代码
> ```matlab
> % 克拉默法则解线性方程组
>
> % 定义方程组
> A = [3 2 -1;
>      2 -2 4;
>      -1 0.5 -1];
>

>
> % 方法2：直接求解（验证）
> x2 = A \ b;
> disp('直接求解:'); disp(x2');
>
> % 方法3：逆矩阵（验证）

> disp('逆矩阵法:'); disp(x3');

## 九、易错点

> [!danger] 常见错误
>
> 1. **浮点误差**：`det(A)` 可能不精确等于 0，用容差判断
> 2. **大矩阵**：高阶行列式数值不稳定，优先用 `rank` 或 `rcond`
> 3. **非方阵**：`det` 只适用于方阵，非方阵会报错
> 4. **符号计算**：精确计算用 `syms`，但速度较慢
