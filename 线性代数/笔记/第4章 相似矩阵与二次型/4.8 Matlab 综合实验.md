## 一、特征值与特征向量

### 1.1 eig 函数

> [!tip] 基本用法
> ```matlab
> e = eig(A)              % 特征值（向量形式）
> [V, D] = eig(A)         % V: 特征向量矩阵, D: 特征值对角矩阵
> [V, D] = eig(A, 'nobalance')  % 不平衡处理（数值稳定性）
> ```

> [!example] 例1：求特征值和特征向量
> ```matlab
> A = [1 2 1; 2 1 1; 1 1 3];
>
> % 方法1：只求特征值
> e = eig(A)
> % → [4, 1, 0] 或其他顺序
>
> % 方法2：求特征值和特征向量
> [V, D] = eig(A)
> % V 的列是特征向量
> % D 的对角元是特征值
>
> % 验证 A*V = V*D
> A * V - V * D    % → 接近零矩阵
> ```

### 1.2 特征多项式

> [!tip] poly 函数
> ```matlab
> p = poly(A)    % 特征多项式系数
> ```

> [!example] 例2：求特征多项式
> ```matlab
> A = [1 2; 3 4];
> p = poly(A)
> % → [1, -5, -2] 表示 λ² - 5λ - 2 = 0
> ```

## 二、相似对角化

### 2.1 判断可对角化

> [!tip] 判断步骤
> ```matlab
> function canDiag = canDiagonalize(A)
> % 判断矩阵是否可对角化
>
>     n = size(A, 1);
>     e = eig(A);
>     unique_e = unique(e);
>
>     for i = 1:length(unique_e)
>         lambda = unique_e(i);
>         algebraic = sum(abs(e - lambda) < 1e-10);  % 代数重数
>         geometric = n - rank(A - lambda * eye(n)); % 几何重数
>
>         if geometric < algebraic
>             canDiag = false;
>             return;
>         end
>     end
>
>     canDiag = true;
> end
> ```

### 2.2 对角化

> [!example] 例3：对角化矩阵
> ```matlab
> A = [4 6 0; -3 -5 0; -3 -6 1];
>
> [V, D] = eig(A);
>
> disp('特征值:'); disp(diag(D)');
> disp('特征向量矩阵 V:'); disp(V);
>
> % 验证
> inv(V) * A * V    % 应接近 D
>
> % 检查 V 是否可逆
> if abs(det(V)) < 1e-10
>     disp('矩阵不可对角化');
> end
> ```

## 三、正交对角化

### 3.1 实对称矩阵正交对角化

> [!example] 例4：正交对角化实对称矩阵
> ```matlab
> A = [2 1 1; 1 2 1; 1 1 2];
>
> % 确认对称
> isSymmetric = norm(A - A') < 1e-10
>
> % 正交对角化
> [Q, D] = eig(A);
>
> % 验证正交性
> orthError = norm(Q' * Q - eye(3))
>
> % 验证对角化
> diagError = norm(Q' * A * Q - D)
> ```

### 3.2 施密特正交化

> [!tip] 自定义正交化函数
> ```matlab
> function Q = gram_schmidt(A)
> % 施密特正交化
> % 输入: A - 列向量为待正交化的向量组
> % 输出: Q - 标准正交化的列向量
>
>     [m, n] = size(A);
>     Q = zeros(m, n);
>
>     for j = 1:n
>         v = A(:, j);
>
>         % 减去在已有正交向量上的投影
>         for i = 1:j-1
>             v = v - (Q(:, i)' * A(:, j)) * Q(:, i);
>         end
>
>         % 单位化
>         if norm(v) > 1e-10
>             Q(:, j) = v / norm(v);
>         end
>     end
> end
> ```

### 3.3 使用 qr 分解

> [!tip] 使用 qr 函数
> ```matlab
> A = [1 1 0; 1 0 1; 0 1 1];
>
> [Q, R] = qr(A);
>
> % Q 的列是标准正交的
> Q' * Q    % → 单位矩阵
> ```

## 四、二次型

### 4.1 构造二次型矩阵

> [!example] 例5：从二次型构造矩阵
> ```matlab
> % f = 2x1² + 3x2² + 4x3² + 2x1x2 - 4x1x3 + 6x2x3
>
> % 方法1：直接构造
> A = [2, 1, -2;
>      1, 3, 3;
>      -2, 3, 4];
>
> % 方法2：从系数构造
> function A = quadraticMatrix(coeffs, n)
> % coeffs: [平方项系数, 交叉项系数]
> % n: 变量个数
>
>     A = zeros(n);
>     idx = 1;
>
>     % 平方项
>     for i = 1:n
>         A(i, i) = coeffs(idx);
>         idx = idx + 1;
>     end
>
>     % 交叉项
>     for i = 1:n-1
>         for j = i+1:n
>             A(i, j) = coeffs(idx) / 2;
>             A(j, i) = coeffs(idx) / 2;
>             idx = idx + 1;
>         end
>     end
> end
> ```

### 4.2 化二次型为标准形

> [!example] 例6：正交变换化标准形
> ```matlab
> A = [2 1 1; 1 2 1; 1 1 2];
>
> % 正交对角化
> [Q, D] = eig(A);
>
> disp('标准形系数（特征值）:'); disp(diag(D)');
>
> % 标准形: f = λ1*y1² + λ2*y2² + λ3*y3²
> % 所用变换: x = Q*y
>
> % 验证
> x = [1; 2; 3];
> y = Q' * x;  % 新坐标
>
> f_old = x' * A * x;  % 原二次型值
> f_new = y' * D * y;  % 标准形值
>
> abs(f_old - f_new) < 1e-10  % 应为 true
> ```

### 4.3 惯性指数

> [!example] 例7：求惯性指数
> ```matlab
> A = [1 2 0; 2 1 0; 0 0 3];
>
> e = eig(A);
>
> p = sum(e > 1e-10);      % 正惯性指数
> q = sum(e < -1e-10);     % 负惯性指数
> z = sum(abs(e) < 1e-10); % 零的个数
>
> fprintf('正惯性指数 p = %d\n', p);
> fprintf('负惯性指数 q = %d\n', q);
> fprintf('秩 r = %d\n', p + q);
> ```

## 五、正定性判定

### 5.1 特征值法

> [!example] 例8：判断正定性
> ```matlab
> A = [2 1 0; 1 2 1; 0 1 2];
>
> e = eig(A);
>
> if all(e > 1e-10)
>     disp('正定');
> elseif all(e < -1e-10)
>     disp('负定');
> elseif all(e >= -1e-10)
>     disp('半正定');
> elseif all(e <= 1e-10)
>     disp('半负定');
> else
>     disp('不定');
> end
> ```

### 5.2 顺序主子式法

> [!example] 例9：顺序主子式法判断正定
> ```matlab
> A = [2 1 0; 1 2 1; 0 1 2];
> n = size(A, 1);
>
> isPositiveDefinite = true;
> for k = 1:n
>     delta = det(A(1:k, 1:k));
>     fprintf('Δ%d = %.4f\n', k, delta);
>     if delta <= 0
>         isPositiveDefinite = false;
>     end
> end
>
> if isPositiveDefinite
>     disp('正定');
> else
>     disp('不正定');
> end
> ```

### 5.3 Cholesky 分解

> [!tip] chol 函数
> ```matlab
> % 正定矩阵可进行 Cholesky 分解
> A = [4 2 2; 2 5 1; 2 1 6];
>
> try
>     L = chol(A, 'lower');  % A = L * L'
>     disp('正定');
>     disp(L);
> catch
>     disp('不正定');
> end
> ```

> [!example] 例10：完整的对角化和二次型分析
> ```matlab

>

>     disp('半正定');

>     disp('不定');
> end
>
