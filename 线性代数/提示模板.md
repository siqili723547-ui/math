# AI笔记生成提示模板

## 使用说明

使用 AI（如 Claudian）根据以下模板生成学习笔记。

---

## ⚠️ 文件命名规范

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| 章笔记 | `第X章 章名.md` | `第1章 矩阵及其运算.md` |
| 节笔记 | `第X章 章名/X.X 节名.md` | `第1章 矩阵及其运算/1.1 矩阵的定义.md` |
| 小节笔记 | `第X章 章名/X.X.X 小节名.md` | `第1章 矩阵及其运算/1.1.1 矩阵的定义.md` |

> ⚠️ **注意**：节笔记直接放在章节文件夹下，不再为小节单独创建子文件夹。

---

## 基础提示词

```
请根据《新工科数学：线性代数》课本，为 [[当前章节]] 生成学习笔记。

要求：
1. 严格定义概念，给出定理的条件和结论
2. 包含计算方法和技巧
3. 给出典型例题和详细解答
4. 注重几何意义的解释
5. 文件命名必须用编号
```

---

## 生成概念类笔记
```
请为《线性代数》生成 "1.1.1 二阶行列式" 的笔记。

要求：
1. 创建文件：线性代数/笔记/第1章 行列式/1.1.1 二阶行列式.md
2. 内容包括：形式化定义、几何意义、性质、例题
3. 使用 callout 格式标注
4. 突出概念之间的联系
```
