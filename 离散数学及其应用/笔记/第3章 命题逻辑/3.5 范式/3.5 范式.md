# 3.5 范式

> [!abstract] 概述
> 范式是命题公式的标准化形式，包括析取范式、合取范式以及主析取范式和主合取范式。范式在公式化简、等价判断和逻辑推理中有重要应用。

## 📌 本节重点

### 核心概念

1. **简单合取式**：由命题变元或其否定组成的合取式
2. **简单析取式**：由命题变元或其否定组成的析取式
3. **小项（极小项）**：包含所有命题变元的简单合取式
4. **大项（极大项）**：包含所有命题变元的简单析取式
5. **主范式**：由小项/大项组成的唯一标准形式

### 范式类型对比

| 类型 | 结构 | 组成元素 | 唯一性 |
|:----:|:----:|:--------:|:------:|
| 析取范式 | 简单合取式的析取 | 简单合取式 | 不唯一 |
| 合取范式 | 简单析取式的合取 | 简单析取式 | 不唯一 |
| 主析取范式 | 小项的析取 | 小项 | 唯一 |
| 主合取范式 | 大项的合取 | 大项 | 唯一 |
