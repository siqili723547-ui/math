## 📌 本节重点

### 核心概念

1. **命题变元**：真值可变的命题符号
2. **合式公式(WFF)**：按递归规则构造的合法公式
3. **解释**：对公式中所有变元的真值指派
4. **真值表**：列出所有解释下公式真值的表格

### 联结词优先级

$$\neg \to \land \to \lor \to \to \to \leftrightarrow$$

**口诀**：非合析蕴等
