1. **命题的概念**：能判断真假的陈述句
2. **五种联结词**：否定、合取、析取、蕴涵、等价
3. **真值表**：判断命题公式类型的工具
4. **等值演算**：基本等值公式的应用
5. **范式**：命题公式的标准形式（主范式）
6. **推理理论**：有效推理的规则和方法
