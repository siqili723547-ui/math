# Claudian 笔记生成提示模板

## 使用说明

在 Claudian 中使用以下提示，让它根据目录结构生成学习笔记。

---

## ⚠️ 文件命名规范（重要）

Claudian 生成笔记时**必须**遵循以下命名规则：

| 层级 | 命名格式 | 示例 |
|------|----------|------|
| 章笔记 | `第X章 章名.md` | `第3章 命题逻辑.md` |
| 节笔记 | `第X章/X.X 节名.md` | `第3章/3.2 命题与命题联结词.md` |
| 小节笔记 | `第X章/X.X.X 小节名.md` | `第3章/3.2.1 命题.md` |

**文件夹结构示例：**
```
离散数学及其应用/
├── 第3章/
│   ├── 3.2 命题与命题联结词.md
│   ├── 3.2.1 命题.md
│   ├── 3.2.2 命题联结词.md
│   └── 3.5 范式.md
```

---

## 基础提示词

```
请根据《离散数学及其应用》课本，为 [[当前章节]] 生成学习笔记。

要求：
1. 按照目录结构填写每个小节的内容
2. 包含定义、定理、公式
3. 给出典型例题和解题思路
4. 标注重点和难点
5. 补充学习技巧和记忆方法
6. 文件命名必须用编号：如 "3.2.1 命题.md"
```

---

## 📋 直接复制使用的指令

### 生成单个小节笔记
```
请为《离散数学及其应用》生成 "3.2.1 命题" 的笔记。

要求：
1. 创建文件：离散数学及其应用/第3章/3.2.1 命题.md
2. 内容包括：定义、性质、例题、易错点
3. 使用 callout 格式标注重点
4. 文件名必须包含编号 "3.2.1"
```

### 批量生成章节笔记
```
请为《离散数学及其应用》第3章生成完整笔记。

要求：
1. 在 "离散数学及其应用/第3章/" 文件夹下创建
2. 每个小节单独一个文件，命名格式：X.X.X 小节名.md
3. 例如：3.2.1 命题.md、3.2.2 命题联结词.md
4. 内容包含定义、定理、例题
```

---

## 详细提示词模板

### 概念类小节
```
请为 [[当前章节]] 的 "X.X.X 小节名称" 生成笔记，包括：
1. 核心定义（用 callout 标注）
2. 重要性质/定理
3. 与其他概念的联系
4. 常见误区
```

### 算法类小节
```
请为 [[当前章节]] 的 "X.X.X 小节名称" 生成笔记，包括：
1. 算法思想
2. 步骤说明
3. 时间/空间复杂度
4. 代码示例（如适用）
5. 例题演练
```

### 应用类小节
```
请为 [[当前章节]] 的 "X.X.X 应用" 生成笔记，包括：
1. 实际应用场景
2. 与计算机科学的联系
3. 具体案例
```

## Callout 格式参考

```markdown
> [!def] 定义
> 定义内容

> [!thm] 定理
> 定理内容

> [!tip] 学习技巧
> 技巧内容

> [!warning] 易错点
> 注意事项

> [!example] 例题
> 例题内容
```
